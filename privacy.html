隱私權政策 / Privacy Policy

最後更新日期 / Last Updated: 2026年3月25日 / March 25, 2026

簡單來說 / TL;DR: 我們不會收集、不會儲存、無法存取您的任何資料。所有資料都只保存在您自己的裝置上。

We do not collect, do not store, and cannot access any of your data. All data is stored only on your own device.

1. 資料收集與使用 / Data Collection and Use

General Sticker System 擴充功能僅在您本地裝置上儲存以下資料：

The General Sticker System extension only stores the following data on your local device:

  • 您添加的貼圖 ID 清單 / Your added sticker ID list
  • 貼圖的標籤與分類資訊 / Sticker tags and category information
  • 您標記的最愛貼圖 / Your favorited stickers
  • 自訂的標籤詞彙 / Custom tag vocabulary

這些資料使用 Chrome 瀏覽器的 chrome.storage.local API 儲存，完全保存在您的裝置上。

This data is stored using Chrome's chrome.storage.local API and is completely stored on your device only.

2. 資料不上傳 / No Data Upload

我們不會 / We do NOT:

  • 將您的資料上傳到任何外部伺服器 / Upload your data to any external server
  • 將資料傳輸給第三方 / Transmit data to third parties
  • 追蹤您的使用行為或瀏覽紀錄 / Track your usage behavior or browsing history
  • 分析或販售您的資料 / Analyze or sell your data

3. 開發者無法存取 / Developer Cannot Access

由於資料僅儲存在您的本地裝置，開發者完全無法存取您的任何貼圖資料或設定。

Since data is only stored on your local device, the developer has absolutely no access to any of your sticker data or settings.

4. 第三方服務 / Third-Party Services

本擴充功能與以下第三方服務互動：

This extension interacts with the following third-party services:

  • Twitch 聊天服務：用於發送聊天訊息 / For sending chat messages
  • Twitch GQL API、Kick API、Vaughn API：用於 TSC（實況編織）功能的開台狀態檢測（僅查詢頻道是否在線，無需登入憑證）/ For TSC stream status checking (online/offline status only, no login credentials required)

我們不會傳輸您的貼圖資料、標籤或任何個人資訊至第三方。發送的內容僅為您手動選擇的貼圖 ID，以及 TSC 功能中您手動設定的公開頻道 URL。

We do not transmit your sticker data, tags, or any personal information to third parties. Only the sticker ID you manually select is sent, along with public channel URLs you manually configure in the TSC feature.

開台狀態查詢結果（在線狀態、觀看人數、實況標題）僅儲存在您的本地裝置快取中（5 分鐘），不會上傳至任何伺服器。/ Stream status results are cached locally only (5 minutes) and never uploaded to any server.

5. 權限使用說明 / Permissions Usage

本擴充功能申請使用以下權限，僅用於所述功能：

This extension requests the following permissions solely for the stated purposes:

all_frames 設定 / all_frames Setting

  • 聊天室 iframe 支援 / Chat Room iframe Support：本擴充使用 all_frames: true 設定，將聊天面板等功能腳本注入 YouTube 等平台的聊天室 iframe 中。這是必要的技術方案，因為 YouTube 聊天室運行在獨立的 iframe 內（網址為 youtube.com/live_chat）。我們不會因此存取或收集任何 iframe 內的個人資料，僅用於顯示貼圖面板。
重要聲明 / Important Note：
本擴充功能絕不紀錄、儲存或傳輸您的瀏覽紀錄或任何分頁 URL 至外部伺服器。所有操作均在本地端即時完成。

This extension NEVER logs, stores, or transmits your browsing history or any tab URLs to external servers. All operations are performed locally in real-time.

6. 聯絡方式 / Contact

如有任何隱私權相關問題，請透過 Chrome Web Store 的開發者資訊頁面與我們聯繫。

For any privacy-related questions, please contact us through the Chrome Web Store developer information page.
