General Sticker System (GSS)

通用貼圖系統

📋 快速開始

使用步驟：

  1. 在 Twitch、YouTube、Vaughn Live、beamstream、w.tv 或 KICK 登入並進入頻道頁面
  2. 開啟插件，輸入貼圖 ID 清單
  3. 點擊聊天室旁的 GSS 圖示開啟貼圖面板
  4. 點擊貼圖快速發送（例如 Twitch/KICK 全自動，其他平台自動填入）

🎨 貼圖格式

🌐 GSS 通用圖片連結

格式：GSS-[圖片連結]

  • 支援格式：.jpg, .jpeg, .png, .gif, .webp, .bmp, .svg, .mp4
  • 支援直接連結或相對連結
  • 範例：GSS-https://example.com/image.jpg
  • 範例：GSS-cdn.discordapp.com/attachments/.../image.png

🔍 連結取得方式：

1. 在網路上看到喜歡的圖片
2. 右鍵複製圖片連結
3. 加上 GSS- 前綴即可使用
※ 支援 Discord、Imgur、或其他任何圖片連結

CatBox 圖片/影片

格式：CB-xxx.ext

支援格式：.gif, .png

範例：CB-z8dta6.gif

🔍 CatBox ID 取得方式：

CatBox 網址範例：https://files.catbox.moe/z8dta6.gif

網址最後一段 z8dta6.gif 就是 ID

Imgur 圖片/影片

格式：IM-xxx.ext

  • 支援格式：.gif, .png, .jpg, .mp4
  • 範例：IM-ha3eTC7.gif、IM-abc1234.mp4

🔍 Imgur ID 取得方式：

Imgur 網址範例：https://i.imgur.com/ha3eTC7.gif 或 https://imgur.com/ha3eTC7

網址最後一段 ha3eTC7 就是 ID
使用時請加上 IM- 前綴和副檔名（如 .gif/.mp4），變成 IM-ha3eTC7.gif
※ 若不知道副檔名，可先嘗試 .gif 或 .mp4

meee.com.tw 圖片/影片

格式：ME-xxx.ext

  • 支援格式：.gif, .png（其他格式待確認）
  • 範例：ME-LXEcxGi.gif、ME-HUi2Bs0.png

🔍 meee ID 取得方式：

meee 網址範例：https://meee.com.tw/LXEcxGi

網址最後一段 LXEcxGi 就是 ID
使用時請加上 ME- 前綴和副檔名（如 .gif/.png），變成 ME-LXEcxGi.gif
※ 若不知道副檔名，可先嘗試 .gif 或 .png

🎬 YouTube 影片

統一格式：YT-xxx（系統自動識別影片類型）

https://www.youtube.com/watch?v=xxxx
https://www.youtube.com/shorts/xxxx

🤖 智能模式：

  • Shorts 短影音（9:16 比例）→ 自動靜音循環播放，可點擊播放
  • 一般影片（16:9 比例）→ 顯示縮略圖，點擊播放
  • 系統自動檢測影片類型
  • 在聊天室輸入會自動顯示縮略圖，點擊開啟內建播放器
  • 範例：YT-tGXZNHNhI0I

🔍 ID 取得方式：

youtube.com/watch?v=xxx → ID 是 xxx → 使用 YT-xxx
youtu.be/xxx → ID 是 xxx → 使用 YT-xxx
youtube.com/shorts/xxx → ID 是 xxx → 使用 YT-xxx

📋 平台支援狀態：

DLive / Twitch / KICK：完整支援縮略圖與播放器
Vaughn Live：YT- 會顯示縮略圖，但完整 YouTube 連結不會轉換（避免官方攔截）

🏷️ 標籤功能

每張貼圖可添加最多 4 個標籤，方便分類和搜尋。

格式：CB-xxx #meme #reaction

  • 標籤以 # 開頭
  • 每個標籤最多 16 個字元
  • 不能包含空格或 #

💡 提示：右鍵點擊貼圖（在頻道頁面或插件面板）即可快速添加/移除標籤

⭐ 快捷操作

  • ★ 常用標記：點擊貼圖上的 ★ 按鈕將其標記為常用，常用貼圖會置頂顯示
  • ✕ 刪除：點擊 ✕ 按鈕從清單中移除貼圖
  • 右鍵標籤：在貼圖上點擊右鍵可添加或移除標籤

🎭 DLive 劇院模式

DLive 設定頁提供多種視覺調整功能：

  • 劇院模式：全屏觀看體驗
  • 聊天室控制：調整聊天室寬度或隱藏
  • 元素控制：隱藏導航欄、標題、送禮區等

🎮 全屏模式快捷鍵：

  • 按 C 鍵：快速開關聊天室
  • 滑鼠移到畫面右側：自動展開聊天室

※ 快捷鍵僅在全屏模式下有效，劇院模式不適用

#https://kick.com/elfinl
#https://vaughn.live/elfinl

🎯 核心功能

  • 多位實況主管理：同時追蹤多位實況主的開台狀態
  • 實時狀態顯示：🟢 在線 / 🔴 離線 / ⚫ 未知
  • 面板控制：點擊網站標誌右側的 ▼ 箭頭展開/收合 TSC 面板，再點擊平台徽章直接跳轉
  • 共用聊天室：例如將 Twitch / KICK 聊天嵌入 DLive / Vaughn 頁面，完全支援 GSS 貼圖（注意：Twitch 聊天因安全策略無法在 KICK 頁面嵌入；KICK 聊天因 CSRF 保護限制，在第三方頁面只能查看無法發言（等待 KICK 官方開放嵌入 API）；Vaughn Live 因無可靠檢測方法，開台狀態暫時顯示 ⚫ 未知）

🧶 TexoStreamCore (TSC) 配信者管理

TSCはGSSの配信者管理プラットフォームで、ライブページで異なる配信者のチャンネルを素早く切り替え、共用チャット機能を使用できます。

📋 設定フォーマット

管理パネルの【編織】タブで入力：

>Elfin
!https://www.twitch.tv/elfinl
#https://www.twitch.tv/elfinl
#https://dlive.tv/ElfinL
#https://kick.com/elfinl
#https://vaughn.live/elfinl
#https://www.youtube.com/@Elfin-Lee

> 配信者名
# プラットフォームURL
! 共用チャット（Twitch / KICK対応）

🎯 主な機能

  • 複数配信者管理：複数の配信者の配信状態を同時に追跡
  • リアルタイム状態：🟢 配信中 / 🔴 オフライン / ⚫ 不明
  • パネルコントロール：サイトロゴの右側にある ▼ 矢印をクリックして TSC パネルを展開/折りたたみ、プラットフォームバッジをクリックして移動
  • 共用チャット：例えばTwitch / KICKチャットをDLive / Vaughnページに埋め込み、GSSステッカーを完全サポート（注意：TwitchチャットはセキュリティポリシーによりKICKページに埋め込めません；KICKチャットはCSRF保護のため、第三者ページでは表示のみで発言できません（KICKの埋め込みAPI公開を待機中）；Vaughn Live は信頼できる検出方法がないため、配信状態は一時的に ⚫ 不明と表示されます）
  • 自動検出：ライブページに #GSS-TexoStreamCore タグを配置すると、拡張機能が自動的に下のデータを読み取ります

🤖 自動検出設定

🧶 TexoStreamCore (TSC) 스트리머 관리

TSC는 GSS의 스트리머 관리 플랫폼으로, 라이브 페이지에서 다양한 스트리머의 채널을 빠르게 전환하고 공용 채팅 기능을 사용할 수 있습니다.

>Elfin

#https://dlive.tv/ElfinL
#https://kick.com/elfinl
#https://vaughn.live/elfinl
#https://www.youtube.com/@Elfin-Lee

# 플랫폼 URL

🎯 핵심 기능

  • 다중 스트리머 관리: 여러 스트리머의 방송 상태를 동시에 추적
  • 실시간 상태: 🟢 방송중 / 🔴 오프라인 / ⚫ 알 수 없음
  • 패널 제어: 사이트 로고 오른쪽의 ▼ 화살표를 클릭하여 TSC 패널을 펼치기/접기, 플랫폼 배지를 클릭하여 이동
  • 공용 채팅: 예를 들어 Twitch / KICK 채팅을 DLive / Vaughn 페이지에 임베드, GSS 스티커 완전 지원 (참고: Twitch 채팅은 보안 정책으로 인해 KICK 페이지에 임베드할 수 없습니다；KICK 채팅은 CSRF 보호로 인해 타사 페이지에서는 조회만 가능하고 발언할 수 없습니다 (KICK 임베드 API 공개 대기 중)；Vaughn Live는 신뢰할 수 있는 감지 방법이 없어 방송 상태가 일시적으로 ⚫ 알 수 없음으로 표시됩니다)
  • 자동 감지: 라이브 페이지에 #GSS-TexoStreamCore 태그를 배치하면 확장 프로그램이 자동으로 아래 데이터를 읽습니다
